RK Driver Assitant
1)Support Rockusb driver and Rockchip Adb driver preinstall and uninstall
2)Support xp,win7_32,win7_64,win8_32,win8_64

Directory Description
Drvier:Rockchip driver file
ADBDriver:Rockchip adb driver file

NOTE:
1)For devices use the updated driver,make sure uninstall driver before install
2)For XP, after install driver completely by the tool,Connect the rockchip device to Windows  
Windows will still detect the device and launch the Found New Hardware wizard. Just select the adb driver folder(ADBDriver) and install


瑞芯微驱动安装助手
支持Rockusb驱动和Rockchip Adb驱动预安装和卸载
支持xp,win7_32,win7_64,win8_32,win8_64操作系统
目录说明：
工具目录\Driver:Rockusb驱动目录
工具目录\ADBDriver:Rockchip ADB驱动目录

注意事项：
为了所有设备都使用更新的驱动，请先“卸载”再进行“安装”
xp系统当通过工具安装驱动完成,连接Rockchip ADB设备时会还提示你发现新设备安装驱动，此时
选择"更新驱动程序"选择Rockchip ADB驱动目录,安装即可。
